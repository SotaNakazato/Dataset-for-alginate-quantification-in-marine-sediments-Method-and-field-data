# Dataset-for-alginate-quantification-in-marine-sediments-Method-and-field-data
This dataset supports the findings of the study titled "Technical note: Development of an extraction protocol and colorimetric analysis for alginate in marine sediment". It provides the primary data required to validate the protocol for quantifying alginate in marine sediments.

-Calibration_data.csv: Detailed experimental data used to generate calibration curves.<img width="468" height="50" alt="image" src="https://github.com/user-attachments/assets/44f479fd-10bd-4237-b264-d96cccadb103" />
-Sampling_Site_data.csv: Coordinates (latitude and longitude) and water depth for each station.<img width="468" height="67" alt="image" src="https://github.com/user-attachments/assets/a5da7f7b-00da-4920-abdd-a1058e673b73" />
-Recovery_conventional_method.csv: Validation data for the conventional method.<img width="468" height="50" alt="image" src="https://github.com/user-attachments/assets/6ad9207d-38e1-47a9-b2cf-960e202441db" />
-Ethanoal_Test.csv: Comparative data evaluating the effect of ethanol precipitation on alginate recovery.<img width="468" height="67" alt="image" src="https://github.com/user-attachments/assets/f798d65d-b39f-47ce-91b0-e6efa86f4380" />
-Recovery _modified_method.csv: Validation data for the modified method.<img width="468" height="50" alt="image" src="https://github.com/user-attachments/assets/f327c23a-97c7-4eee-b642-a3a670588b8c" />
-Phytoplankton_Test.csv: Experimental data evaluating potential interference from phytoplankton samples.<img width="468" height="67" alt="image" src="https://github.com/user-attachments/assets/805ef267-4a0c-4401-9841-9236a11aea5b" />
-Enzymatic_degradation_Test.csv: Data from enzymatic degradation experiments used to confirm that the extracted uronic acids are derived from alginate.<img width="468" height="67" alt="image" src="https://github.com/user-attachments/assets/80aa7de8-7cef-471f-93f4-c6ea1b463eec" />
-Alginate_amount_Hokkaido_2026.csv: Quantification results of alginate concentrations in surface marine sediments collected around Hokkaido, Japan.<img width="468" height="67" alt="image" src="https://github.com/user-attachments/assets/687e6fd6-1139-4a84-8675-7006b28eb6a5" />
